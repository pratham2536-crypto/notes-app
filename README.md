# Journal App

A beautiful, modern, and distraction-free note-taking and journaling web application built with pure Vanilla HTML, CSS, and JavaScript.

## Features
- 🌙 **Premium Dark Mode**: Sleek UI with glassmorphism elements.
- 🎯 **Focus Mode**: Hide the sidebar for a pure distraction-free writing experience.
- ✨ **Dynamic Parallax Background**: The background subtly shifts as you move your mouse.
- 💾 **Local Storage**: Auto-saves your notes instantly into your browser's secure local storage.
- 📊 **Live Stats**: Real-time word and character counting.

## Getting Started

Because this application relies on Vanilla Web Technologies, no installation or build steps are required.

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/journal-app.git
   ```
2. Navigate to the project folder:
   ```bash
   cd journal-app
   ```
3. Open `index.html` in your favorite web browser:
   ```bash
   # On macOS
   open index.html
   ```
